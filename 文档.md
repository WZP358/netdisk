### 在 NameNode Web UI 中浏览文件

打开上述地址后，按照以下步骤操作：

1. 在主页面顶部菜单栏，点击 "Utilities" → "Browse the file system"



1. 或者直接访问：

- Hadoop 3.x: http://192.168.88.161:9870/explorer.html

- Hadoop 2.x: http://192.168.88.161:50070/explorer.html

1. 浏览到你的项目文件目录：/netdisk









### 

\# 查看 HDFS 根目录

hdfs dfs -ls /

\# 查看你的项目文件目录

hdfs dfs -ls /netdisk

\# 递归查看所有子目录和文件

hdfs dfs -ls -R /netdisk

\# 以人类可读的格式显示文件大小（带单位）

hdfs dfs -ls -h /netdisk

### 2️⃣ 查看文件内容

\# 查看文件内容（类似 cat）

hdfs dfs -cat /netdisk/文件名

\# 查看文件头部（前几行）

hdfs dfs -head /netdisk/文件名

\# 查看文件尾部（最后几行）

hdfs dfs -tail /netdisk/文件名

\# 查看大文件（分页查看）

hdfs dfs -cat /netdisk/文件名 | more

\# 或

hdfs dfs -cat /netdisk/文件名 | less

### 3️⃣ 查看文件详细信息

\# 查看文件统计信息（大小、副本数等）

hdfs dfs -stat "%n %b %r" /netdisk/文件名

\# %n = 文件名

\# %b = 文件大小（字节）

\# %r = 副本数

\# 查看目录磁盘使用情况

hdfs dfs -du /netdisk

\# 以人类可读的格式查看

hdfs dfs -du -h /netdisk

\# 递归统计目录大小

hdfs dfs -du -s -h /netdisk

\# 查看磁盘使用情况汇总

hdfs dfs -df -h /

### 4️⃣ 查看文件的块信息（存储位置）

\# 查看文件存储在哪些 DataNode 上

hdfs fsck /netdisk/文件名 -files -blocks -locations

\# 查看整个目录的健康状况

hdfs fsck /netdisk

### 5️⃣ 搜索文件

\# 查找特定文件

hdfs dfs -find /netdisk -name "*.txt"

\# 或使用 ls 配合 grep

hdfs dfs -ls -R /netdisk | grep "文件名"

### 6️⃣ 查看 HDFS 集群状态

\# 查看 HDFS 集群报告

hdfs dfsadmin -report

\# 查看 NameNode 状态

hdfs dfsadmin -safemode get

\# 查看所有 DataNode

hdfs dfsadmin -printTopology

### 7️⃣ 实用示例组合

\# 查看文件数量统计

hdfs dfs -count /netdisk

\# 输出格式：目录数 文件数 总大小 路径

\# 按时间排序查看文件（配合 ls 的输出）

hdfs dfs -ls /netdisk | sort -k6,7

\# 查找最近修改的文件

hdfs dfs -ls -R /netdisk | grep "^-" | awk '{print $6,$7,$8}' | sort -r | head -10

### 8️⃣ 下载文件到本地查看

\# 下载 HDFS 文件到本地

hdfs dfs -get /netdisk/文件名 /本地路径/

\# 复制整个目录

hdfs dfs -get /netdisk /本地路径/

\# 合并下载（将多个小文件合并成一个）

hdfs dfs -getmerge /netdisk/*.txt /本地路径/merged.txt

### 9️⃣ 常用快捷别名（可选设置）

你可以在 ~/.bashrc 中添加别名，方便使用：

\# 编辑 bashrc

vi ~/.bashrc

\# 添加以下内容

alias hls='hdfs dfs -ls'

alias hlsr='hdfs dfs -ls -R'

alias hcat='hdfs dfs -cat'

alias hdu='hdfs dfs -du -h'

alias hget='hdfs dfs -get'

\# 保存后执行

source ~/.bashrc

\# 之后就可以简化使用

hls /netdisk

hcat /netdisk/test.tx
